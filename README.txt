Dự án bao gôm các chức năng sau:
- Ở giao diện trang chủ có hiển thị sản phẩm theo category, thêm sản phẩm vào giỏ hàng. Đăng nhập, đăng xuất, đăng kí người dùng, thông tin người dùng được lưu vào database dưới quyền user, quyền admin được thêm trực tiếp ở mục role ở bảng users trong database.
- đăng nhập giao diện admin ở đường dẫn /admin/login.jsp. Tài khoản admin được tạo bằng cách đăng kí users ở trang chủ rồi set quyền admin trong database. Trong này có thể thêm sửa xóa tìm kiếm sản phẩm, category, thay đổi trạng thái đơn hàng.
